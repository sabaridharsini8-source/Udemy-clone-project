# Udemy-clone-project
This is a Udemy clone website built using HTML and basic CSS. It replicates the look and feel of the Udemy platform and demonstrates a clean layout for online courses.

Features

Simple course listing layout with images and titles

Navigation bar with menu items

Responsive frontend design using HTML and CSS only

Technologies Used

HTML5 – for website structure

CSS3 – for styling and layout

Purpose

This project is a frontend practice project to improve skills in HTML and CSS by creating a real-world website clone.
